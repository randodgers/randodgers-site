# Deploy randodgers-site on GoDaddy (Multi-page)

Upload all files from this folder to your web root:
- index.html
- templates.html
- services.html
- about.html
- contact.html
- privacy.html
- terms.html
- 404.html
- styles.css
- favicon.svg
- og-image.svg
- robots.txt
- sitemap.xml

## Domain structure
- randodgers.com -> main site (this folder)
- www.randodgers.com -> redirect to randodgers.com
- app.randodgers.com -> future app/login
- api.randodgers.com -> future backend/webhooks
